# 알비온 온라인 길드 생활 가이드

## 1. 장비 시스템

### 티어 (Tier) 체계
알비온의 장비는 T2~T8까지 7단계 티어로 구분된다.

| 티어 | 명칭 | 기준 IP |
|------|------|---------|
| T2 | Novice | - |
| T3 | Journeyman | - |
| T4 | Adept | 800 |
| T5 | Expert | 1000 |
| T6 | Master | ~1200 |
| T7 | Grandmaster | ~1400 |
| T8 | Elder | ~1500 |

> T4.0 = 800 IP, T5.0 = 1000 IP, T8.4(최고) = 1900 IP (품질 제외 기준)

### 인챈트 (Enchantment)
T4 이상 아이템에 적용 가능. 인챈트마다 IP +100.

| 단계 | 희귀도 명칭 |
|------|------------|
| .0 | 기본 (Normal) |
| .1 | 언커먼 (Uncommon) |
| .2 | 레어 (Rare) |
| .3 | 익셉셔널 (Exceptional) |
| .4 | 프리스틴 (Pristine) |

- 예: T4.3 = 1100 IP (T6 수준과 유사)

### 품질 등급 (Quality)
| 등급 | IP 보정 |
|------|---------|
| Normal | 기본 |
| Good | +20 |
| Outstanding | +50 |
| Excellent | +100 |
| Masterpiece | +200 |

## 2. 방어구 (Armor)

방어구는 **천(Cloth) / 가죽(Leather) / 판금(Plate)** 세 종류로 나뉜다.  
각각 머리 / 몸통 / 발 3부위로 구성된다.

### 천 방어구 (Cloth Armor)
마법 저항(MR)이 높고, 물리 방어는 낮다. 고화력·치유 빌드 전용.

| 부위 | 아이템 | 주요 액티브 | 역할 |
|------|--------|------------|------|
| 몸통 | 로브 (Robe) | Mend Wounds / Frost Shield / Speed Caster | 힐러·DPS |
| 머리 | 카울 (Cowl) | Energy Regain / Force Field / Energy Shield | 에너지 관리 |
| 발 | 샌들 (Sandals) | Dodge / Energetic Sprint / Focused Run | 기동성 |

**주요 로브 종류:**  
Scholar Robe, Cleric Robe, Druid Robe, Mage Robe, Cultist Robe, Hellion Robe, Specter Robe, Fiend Robe, Undead Robe 등

**공통 패시브:** Aggression / Concentration / Efficiency (Sandals는 Courier 추가)

---

### 가죽 방어구 (Leather Armor)
물리·마법 방어가 균형적. 이동성 및 지속전 특화.

| 부위 | 아이템 | 주요 액티브 | 역할 |
|------|--------|------------|------|
| 몸통 | 재킷 (Jacket) | Haste / Ambush / Inferno Shield 등 | 근거리 DPS·서포터 |
| 머리 | 후드 (Hood) | - | 정찰·DPS |
| 발 | 신발 (Shoes) | Refreshing Sprint | 기동성 |

**주요 재킷 종류:** Hunter Jacket, Mercenary Jacket, Assassin Jacket, Royal Jacket + 아티팩트  
- **Hunter Jacket:** Haste (이동속도 50%·공격속도 60% 8초)  
- **Assassin Jacket:** Ambush (은신 후 기습), Mend Wounds, Inferno Shield  
- **Mistwalker Jacket:** 클랩 DPS의 핵심 아이템 (ZvZ 메타)

---

### 판금 방어구 (Plate Armor)
물리 방어(Armor)가 높고, MR은 낮다. 고생존·군중제어(CC) 특화. 탱커 전용.

| 부위 | 아이템 | 주요 액티브 | 역할 |
|------|--------|------------|------|
| 몸통 | 갑옷 (Armor) | Enfeeble Aura / Taunt | 적 약화·도발 |
| 머리 | 헬멧 (Helmet) | Block / 강제이동 면역 | 생존·CC저항 |
| 발 | 부츠 (Boots) | Giant (최대 HP·이동속도 증가) | 생존성 |

**주요 갑옷 종류:** Soldier Armor, Knight Armor, Guardian Armor, Royal Armor + 아티팩트  
- **Guardian Armor:** Enfeeble Aura (적 피해 감소·슬로우·투명 해제), 가장 안정적인 PvE 탱 갑옷  
- **Soldier Helmet:** Block (일시적 피해·CC 무적)  
- **Knight Helmet:** 강제이동 효과 면역 — ZvZ에서 필수

## 3. 무기 (Weapons)

### 무기 계열 분류
알비온 무기는 **전사 / 사냥꾼 / 마법사** 3개 계열, 총 20개 트리.

#### 전사 (Warrior)
| 무기 | 특성 |
|------|------|
| 도끼 (Axe) | 출혈 DoT, 근접 지속딜 |
| 검 (Sword) | 다목적 근접, 이동기 보유 |
| 메이스 (Mace) | CC, 탱킹 보조 |
| 망치 (Hammer) | 광역 CC, ZvZ 탱커 핵심 |
| 전투 글러브 (War Gloves) | 고속 근접, 독특한 콤보 |
| 석궁 (Crossbow) | 근거리 원딜, 버스트 특화 |
| 방패 (Shield) | 오프핸드 전용, 방어·반격 |

#### 사냥꾼 (Hunter)
| 무기 | 특성 |
|------|------|
| 활 (Bow) | 원거리 지속딜, 기동성 |
| 단검 (Dagger) | 은신·암살, 단독 플레이 |
| 창 (Spear) | 광역 CC, 그룹 딜러 |
| 쿼터스태프 (Quarterstaff) | 근접 CC, 기동성 |
| 변형자 지팡이 (Shapeshifter Staff) | 변신 기믹, 복합 역할 |
| 자연 지팡이 (Nature Staff) | 지속 힐, 그룹 서포트 |
| 횃불 (Torch) | 오프핸드, 공격속도 증가 |

#### 마법사 (Mage)
| 무기 | 특성 |
|------|------|
| 화염 지팡이 (Fire Staff) | 고화력 AoE, 스킬샷 |
| 성스러운 지팡이 (Holy Staff) | 힐, 단일·광역 치유 |
| 비전 지팡이 (Arcane Staff) | 에너지 드레인, 서포트 |
| 서리 지팡이 (Frost Staff) | CC, 슬로우·빙결 |
| 저주받은 지팡이 (Cursed Staff) | DoT, 생명력 흡수 |
| 마법서 (Tome of Spells) | 오프핸드, 에너지·방어 |

### 무기 특성
- **1손 무기:** 오프핸드(방패·횃불·마법서 등) 장착 가능
- **2손 무기:** 두 손 사용, 더 강력한 E스킬 보유
- **일반 무기:** 자원으로 제작
- **아티팩트 무기:** 희귀 재료 필요, 고유 스킬 보유

## 4. 데스티니 보드 (Destiny Board)

캐릭터 성장 시스템. 명성(Fame)을 획득해 잠금 해제.

- **전사 계열:** 판금 방어구 + 도끼·검·망치·메이스·석궁·방패
- **사냥꾼 계열:** 가죽 방어구 + 활·단검·창·자연 지팡이·쿼터스태프·횃불
- **마법사 계열:** 천 방어구 + 화염·서리·성스러운·비전·저주·마법서

### 마스터리 (Mastery)
특정 장비를 반복 사용해 레벨 획득. 최대 100레벨 → Elite Level로 전환.  
마스터리가 높을수록 해당 아이템의 실질 IP 상승.

## 5. 길드 조합 (Guild Composition)

### 규모별 구분
| 규모 | 인원 기준 |
|------|----------|
| 스몰 스케일 | 20인 이하 |
| 미들 스케일 | 40인 내외 |
| ZvZ (제르그 vs 제르그) | 60인 이상 |

### 역할 종류

#### 탱커 (Tank)
- 이니시에이터형: 적에게 먼저 CC를 걸고 교전 시작
- 수비형: 아군 힐러·서포터 보호, 적 CC 방해
- **주요 무기:** 망치(Hammer), 메이스(Mace), 그로브키퍼(Grovekeeper), 저스티스의 손(Hand of Justice)

#### 힐러 (Healer)
- 광역 힐로 아군 생존 유지
- **주요 무기:**
  - **폴른 스태프 (Fallen Staff):** ZvZ 필수픽. Salvation(E) = 광역 버스트 힐 + CC·디버프 전체 해제
  - **자연 지팡이 (Nature Staff/Wild Staff):** 지속 힐 장판, 폴른과 함께 사용
  - **할로우폴 (Hallowfall):** 소규모 PvP·힐게이트 특화

#### DPS (딜러)
- ZvZ에서는 **광역기** 보유 무기만 기용
- **근접 DPS (브롤 계열):** 반지름 내 다수 타격, Bloodletter, Carrioncaller, Halberd, Hellfire Hands, Spiked Gauntlets, Infernal Scythe
- **원거리 DPS (클랩 계열):** Longbow, Siegebow, Wailing Bow, Permafrost Prism, Energy Shaper, Witchwork Staff
- **폭탄 역할:** Rift Glaive — 적 후방 침투·폭탄 투척 특화

#### 서포터 (Support)
- 아군 보조, CC, 에너지 공급
- **주요 무기:** Icicle Staff, Malevolent Locus, Arcane Staff 계열
- 아케인 스태프 계열 외에도 딜러·탱커 무기 중 보조 역할로 분류되는 경우 있음

#### 칼러 (Caller)
- 교전 지휘자. 타겟 지정 및 교전 진입·이탈 명령
- 무기보다 판단력과 커뮤니케이션이 핵심

#### 스카우트 (Scout)
- 적 동향 파악, 정찰 전담
- 고기동성 빌드 사용 (은신·이동기 특화)

## 6. 주요 조합 유형

### 브롤 (Brawl)
- **특성:** 지속딜 중심. 꾸준히 누적 피해로 적을 소진시키는 방식
- **특징:** 아군과 뒤섞여 근접전, 지속 교전 유리
- **핵심 역할:** 근접 DPS 다수 + 폴른 힐러

### 클랩 (Clap)
- **특성:** 버스트·범위기 중심. 순간적인 폭발 피해로 일거에 제압
- **특징:** Mistwalker Jacket이 핵심 아이템 (은신→기습)
- **핵심 역할:** 원거리/폭탄 DPS + 서포터

> 대규모 ZvZ에서는 브롤 파티 + 클랩 파티를 병행 운용하는 혼합 조합 사용.

## 7. 소모품 (Consumables)

### 음식 (Food)
시간 경과에 따른 지속 버프 제공. 전투 전 항상 섭취.

| 종류 | 주요 효과 |
|------|----------|
| 스튜 (Stew) | 전투 피해 증가, 쿨다운 감소, HP 재생 |
| 샌드위치 (Sandwich) | 최대 HP 증가, 받는 힐량 증가, CC 지속시간 증가 |
| 로스트 (Roast) | 피해 흡혈, 최대 HP 증가 |
| 오믈렛 (Omelette) | 시전속도, 쿨다운, 방어력 증가, 에너지 비용 감소 |

### 포션 (Potions)
즉각 효과. 음식 버프와 별개로 사용 가능. 단축키 `1`로 사용.

| 종류 | 주요 효과 |
|------|----------|
| HP 포션 | 즉각 체력 회복 또는 HoT |
| 에너지 포션 | 에너지(마나) 즉각 회복 |
| 방어 포션 | 일시적 방어력 증가 |
| 기타 | 속도 포션, 은신 감지 포션 등 |

## 8. 오프핸드 아이템 (Off-Hand)

1손 무기 사용 시 왼손에 장착하는 보조 아이템.

| 종류 | 주요 효과 |
|------|----------|
| 방패 (Shield) | 방어·반격, 탱커 필수 |
| 횃불 (Torch) | 공격속도 증가, 자동공격 빌드 특화 |
| 마법서/책 (Tome/Book) | 에너지·피해 보조, 마법사 계열 |

**주요 횃불 종류:** Mistcaller, Leering Cane, Cryptcandle, Sacred Scepter  
**주요 마법서 종류:** Eye of Secrets, Muisak, Taproot, Celestial Censer
